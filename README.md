🧮 Matrix Operations Tool using NumPy
📌 Project Overview

This is an interactive Python-based Matrix Operations Tool built using the NumPy library. It allows users to input matrices and perform common linear algebra operations through a simple menu-driven interface.

⚙️ Features
➕ Matrix Addition
➖ Matrix Subtraction
✖ Matrix Multiplication
🔄 Matrix Transpose
📐 Determinant Calculation
🧭 Interactive Menu System
🧠 Technologies Used
Python
NumPy
🚀 How to Run the Project
1. Install NumPy
pip install numpy
2. Run the program
python matrix_tool.py
📥 Input Format

When prompted, enter matrix size like:

2 2

Then enter matrix values row by row:

1 2
3 4
🧾 Example Usage
Input:

Matrix A:

2 2
1 2
3 4

Matrix B:

2 2
5 6
7 8
Output Operations:
A + B
A - B
A × B
Transpose(A)
Determinant(A)
